# AdventOfCode
my solutions for the [Advent of Code](https://adventofcode.com/2022) written in Java.

this will be my first year participating (2022), and since i haven't used java in a while, it's the perfect opportunity to get some practice.

i'll be using JUnit 5 to check against the examples and some very lightweight interfaces / classes i prepared to have a basic structure for the daily puzzles (like reading the input data, providing the test results etc.)

## Structure
Since i want to have the solutions for several years in one repo, you'll find the single Solutions in `src/main/java/li/ste/adventofcode/year[year]/Day[number].java`. The data for the puzzles can be found similarly: `src/main/resources/[year]/Day[number].txt`
